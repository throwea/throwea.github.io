# throwea.github.io

Personal portfolio site for Elian Ahmar. Built with SvelteKit, TailwindCSS, and deployed to GitHub Pages.

## Development

```bash
./dev.sh          # dev server at localhost:5173
./dev.sh preview  # build + preview at localhost:4173
./dev.sh build    # build static output to ./build
```

Requires Node 22+.
